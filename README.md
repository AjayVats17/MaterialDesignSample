Material Design Topics Sample
===================================

>This project is a sample for Matrial Design:

- Floating Button
- Navigation Drawer
- Collapsing Toolbar
- Co-ordinate layout
- Slide Transition
- Ripple Effect
- Reveal Animation
- Custom Snack Bar
- Tab Layout
- Swipe Refresh
- Zoom In Zoom Out Animation
- Other Simple Animation



Pre-requisites
--------------

- Android SDK v26
- Android Build Tools v26.0.0
